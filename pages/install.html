WSL Installation

Every Wordpress user should know how to install plugins, but just in case here is how to install WSL.

1. Install WSL the hard way, Seriously tho, Who on earth would do that!

  1. Download, Unzip and drop the extention on /wp-content/plugins/ directory,
  2. As admistration, activate the plugin through the 'Plugins' menu in WordPress,
  3. Goto the Settings > WP Social Login to get started.

2. Install WSL the easy way, Click Click Click.

  1. As admistration, goto 'Plugins' then Click on 'Add New',
  2. Search for 'WordPress Social Login' then Click on 'Install Now',
  3. Wait for it to download, Unpack and to install,
  4. Activate the plugin by Clicking on 'Activate Plugin'
  5. Goto the Settings > WP Social Login to get started.
